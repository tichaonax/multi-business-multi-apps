# Images Directory

## Eco-Cash Logo

Please place your Eco-Cash logo image as:
- **Filename**: `ecocash-logo.png`
- **Location**: This directory (`public/images/`)
- **Recommended size**: 200x80 pixels (or similar aspect ratio)
- **Format**: PNG with transparent background

The logo will be displayed on the customer display next to the phone number when the "Accepts Eco-Cash" checkbox is enabled in the business settings.
